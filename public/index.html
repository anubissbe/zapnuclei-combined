About This API

This is a minimal Node.js Express API designed for demonstrating security scanning workflows. It provides basic endpoints for health monitoring and service status reporting.

Features

  • Security headers implementation
  • JSON API responses
  • Error handling middleware
  • Health monitoring endpoints

Integration

This API is designed to work with:

  • OWASP ZAP - Web application security testing
  • Nuclei - Vulnerability scanning
  • GitHub Actions - CI/CD security workflows
  • SARIF - Security results interchange format
